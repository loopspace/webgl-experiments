Voronoi Explorer

This page allows you to explore generalised Voronoi diagrams.

There are two main modes:

  • Multiple points: in this mode, there are 16 points which can be dragged around. The parameters are set at random.
  • Double points: in this mode, there are 2 points which are at fixed locations. The parameters can be set manually.

The generalisation is to allow weights and delays. These modify the distance function to be of the form:

ax+ba x + b

where aa (the weight) and bb (the delay) can vary with the source point.

In double point mode, you set these for one of the points since it is the relative weight and relative delay that counts.
